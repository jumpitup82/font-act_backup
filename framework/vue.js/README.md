### 탄생 배경

* 2013년 google 엔지니어 Evan You가 개발
* AngularJS에서 좋아하는 특성만 담은 가벼운 라이브러리를 만들고자 탄생
* 2016년 4월 V2.0 발표 이후 인기 얻기 시

### 장점

* 배우기 쉽다.
* 다른 라이브러리나 기존 프로젝트와의 통합이 쉽다.
* Virtual DOM을 사용하여 일반적인 상황에서 충분히 빠르다.
